# 💰 SLL 记账 Pro (SLL Finance Pro)

> **“让每一分钱都有迹可循，让财务未来触手可及。”**
>
> **SLL 记账 Pro** 是一款由孙礼龙（笔名：孑余庆）开发的现代化 Android 财务管理应用。它不仅提供了极致丝滑的 UI 交互，更在逻辑层实现了从“碎片记录”到“宏观统计”的深度闭环。

---

## ✨ 核心深度特性

### 1. 维度化的统计引擎

不同于传统的记账软件，本项目内置了**时间维度聚合算法**：

- **月度看板**：自动分析当月收支比，精准计算净结余。
- **年度总结**：站在 365 天的高度审视消费趋势，帮助用户建立长期理财观念。
- **动态色彩反馈**：应用会根据“盈余”或“赤字”自动调整 UI 色调，通过视觉心理学提醒用户控制预算。

### 2. 声明式 UI 架构

全量采用 **Jetpack Compose** 构建，告别冗余的 XML：

- **状态驱动**：UI 随 `FinanceState` 实时刷新。
- **丝滑过渡**：使用 `Crossfade` 和 `HorizontalPager` 实现多页面间的无缝平滑切换。
- **响应式布局**：完美适配不同比例的 Android 机型。

### 3. 智能标签与色彩逻辑

- **自动取色**：新生成的自定义标签会通过哈希映射自动匹配系统调色盘中的协调颜色。
- **沉浸式交互**：在添加账单时，支持快速滑动选择标签及日历即时选择。

## 🛠 技术规格与架构

### 技术栈

- **核心语言**：Kotlin 1.9+
- **UI 框架**：Jetpack Compose (Material 3)
- **异步处理**：Kotlin Coroutines
- **持久化**：SharedPreferences (基于 String 序列化的轻量级对象存储方案)
- **生命周期管理**：CompositionLocal 与 Remember 状态持久化

### 项目结构

Plaintext

```
com.sll.finance
├── MainActivity.kt        # 程序主入口、导航逻辑与多层视图切换
├── ThemeColors.kt         # 规范化的全局色彩管理与标签配色引擎
├── FinanceRepo.kt         # 数据持久化层，处理用户、账单、标签的读写
├── FinanceState.kt        # 业务逻辑层，管理状态、增删改查及数据统计算法
└── Components/            # 自定义 UI 组件（如饼图、统计卡片、对话框等）
```

## 🏗 安装与配置

### 开发环境

- **Android Studio**: Jellyfish (2023.3.1) 或更新版本。
- **Gradle**: 8.0+。
- **最小支持版本**: Android 7.0 (API Level 24)。

### 构建指南

1. 修改 `build.gradle.kts` 中的版本信息（当前版本：v3.2）。
2. 在 `MainActivity.kt` 中可自行调整默认测试数据（InjectData 逻辑）。
3. 使用 **Build > Build APK(s)** 生成安装包。

## 📖 使用说明

1. **初次使用**：
   - 使用内置管理员账号 `admin` (密码 `123456`) 快速预览功能。
   - 或者通过注册界面创建属于自己的本地账户。

2. **记账**：点击右下角 `+` 号，输入金额，点击日历选择日期（默认为今天），选择分类标签即可。
3. **统计查看**：向右滑动或点击“统计”标签。点击顶部的“月度/年度”按钮可切换不同的统计口径。
4. **数据维护**：长按或点击明细条目可进行编辑或软删除（进入回收站）。
